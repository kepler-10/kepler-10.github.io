#### 1.how to change git remote url ?

#### 2.本地创建新分支并push到远程

```xml
查看分支：git branch -a

1.本地创建新分支
git branch -c `branch-name`
2.将本地分支push到远程，并建立远程分支
git push -u origin `branch-name` 
```
